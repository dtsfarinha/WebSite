Este Website é um projeto de faculdade cujo o objetivo era aprender as bases de front end e back end e assimilar conhecimentos em Webdesign. Para este projeto usou-se a linguagem Html em conjunto com Css, Python e o framework Flask, um pouco de JavaScript em especial menção o "sistema" Skulpt que é essencialmente um compilador que permite correr Python num browser. Usados ainda no lançamento do site foram o git e heroku (servidor). A formatação do site foi feita com um layout de Grid composto por um header, um main e um footer e em versões anteriores um nav que agora se encontra implementado no header.

Na página algoritmos encontram-se 4 programas implementados no back end do site cada um relacionado com a teoria dada na cadeira de teoria da computação.
O Autómato Finito requisita uma sequência para ser processada (sequência de "a"s e "b"s) numa tabela de transições de estados (no código um dicionário) onde vai avançar de estado para estado começando no inicial e acabando com sucesso no final.
Já o Autómato com pilha funciona com uma pilha onde é depositada um "$" que vai sinalizar o final do processo em sucesso. Igualmente ao autómato finito, este autómato requisita uma sequência ao utilizador, mas desta vez de 0's e 1's onde o número de 0's é igual ao de 1's e todos os 0's aparecem primeiro. Ao ler a sequência por cada 0 que lê, o algoritmo vai colocar um marcador "x" na pilha e por cada 1 que lê vai retirar esse marcador até não exisir mais nada na sequência e restar apenas o simbolo "$".
O Regex (short for regular expression) é uma sequência de caracteres que vai definir um padrão, neste caso uma password e um Email. A password só vai ser aceite se tiver no mínimo uma minúscula, uma maiúscula, um número e pelo menos 8 caracteres e o Email terá que seguir as regras normais para criar um Email.
Por fim o ultimo algoritmo, a turing machine visa organizar uma sequência de 0's e 1's usando uma fita abstrata, um conjunto de estados pré-programado e uma sequência dada pelo utilizador. O algoritmo começa então por alocar a sequência na fita e colocando um "$"no final; depois vai encontrar todos os 0's do número, substituindo-os por "x" e recolocando-os depois do "$"; a seguir fará o mesmo para os "1"s colocando-os depois de todos os "0"s seguindo de limpar todos os "x"s e o "$" acabando assim com uma sequência organizada.

Na última página do site encontrasse o projeto final da cadeira: Game Of Life. Este jogo é um autómato celular que tem o objetivo de reproduzir através de regras simples o desenvolvimento de grupos de seres vivos. A aplicação é "Turing Complete" o que significa que tem a capacidade de emular uma máquina de turing, isto é que o sistema é capaz de reconhecer e decidir a manipulação de data consoante as regras que lhe são dadas. Este jogo é constituído por um tabuleiro quadriculado em que as células podem estar vivas(1) ou mortas(0) e seguem as seguintes regras:
1)Cada célula com menos de duas vivas adjacentes morre por, subpopulação
2)Uma célula com duas ou três vivas adjacentes continua viva
3)Qualquer célula com mais de 3 adjacentes vivas morre por, sobrepopulação
4)Uma célula morta com 3 vivas adjacentes torna se viva(reprodução)

Para a criação desta aplicação havia algumas escolhas em termos de tecnologias viáveis como pygame ou bibliotecas com matplotlib mas decidimos experimentar algo novo e diferente utilizando "Skulpt" que permite correr python no browser como JavaScript. Para desenhar os gráficos usamos o Turtle (biblioteca gráfica de python) mas devido à passagem pelo Skulpt perdeu se algum performance.
