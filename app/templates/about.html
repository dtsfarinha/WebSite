O game of life é um autómato celular e tem o objetivo de reproduzir através de regras simples o desenvolvimento de grupos de seres vivos. A aplicação é "Turing Complete" o que significa que tem a capacidade de emular uma maquina de turing, isto quer dizer que o sistema é capaz de reconhecer e decidir a manipulação de data consoante as regras que lhe são dadas.
O funcionamento é constituido por um tabuleiro quadriculado em que as celulas podem estar ou vivas(1) ou mortas(0) e seguem as seguintes regras
1)Cada celula com menos de duas vivas adjacentes morre por, subpopulação
2)Uma celula com duas ou três vivas adjancentes continua viva
3)Qualquer celula com mais de 3 adjancentes vivas morre por, sobrepopulção
4)Uma celula morta com 3 vivas adjancentes torna se viva(reprodução)

Para a criação desta aplicação haviam algumas escolhas em termos de tecnologias viaveis como pygame ou bibliotecas com matplotlib mas decidimos escolher experimentar algo novo e diferente utilizando "Skulpt" que permite correr python no browser como JavaScript. Caso o tema do trabalho o permitisse teria sido muito mais eficiente faze-lo em JavaScript aproveitando tecnologias como o React para o front-end. Para desenhar os graficos usamos o Turtle(biblioteca grafica do python) mas devido à passagem pelo Skulpt perdeu se algum performance.
O resto do Website foi feito com HTML, CSS, layout CSS Grid, website dinâmico e responsivo, Python, JavaScript (Skulpt), Flask, Git e Heroku.
